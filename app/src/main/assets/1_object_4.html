Фиолетовые пакеты исследований

 

Также для перевозки ресурсов на большие расстояния нужно много рельс, по данной схеме мы можем сделать завод по производству рельс.

 

Совмещая между собой производства можно создать фиолетовые пакеты главное следить за тем, чтобы на производстве не было недостатка ресурсов, иначе это растянется надолго.
